背景

这个动画的背景是在做一个专业课介绍网站时，有一个动画因flash已不再支持而无法显示。恰好又因为在逛某404网站时，看到了The Coding Train的Code! Programming with p5.js系列，（加之刚好考完研了，有点闲了）于是就想到了用p5js来重现这个flash动画的效果。

实现过程

动画拆解

通过观察，可以看到本动画主要包含以下几个部分：

  1. 一个固定的背景
  2. 蓝色激光源在动画刚开始时需要有渐入效果
  3. 试管中四种粒子的不规则向上运动
  4. 流动腔四种粒子的依次向下流出，且在动画刚开始时有蓝色背景逐渐向下移动效果
  5. 蓝色激光源在检测到粒子时，需要显示对应的蓝色和绿色激光，并在电脑上显示检测到的粒子

素材提取

由于原始动画是使用flash制作的，故首先使用jpexs-decompiler提取了所需要的图形文件，并将其保存成svg矢量图格式方便后续进行缩放。

背景制作

静态的背景较为简单，直接使用ps、ai、在线svg编辑器啥的合成一张大背景，随后直接显示即可

蓝色激光源的渐入效果

此动画的实现也较为简单，使用tint()函数可修改显示图片的不透明度，维护alpha变量，使其在每次更新时增加easing值即可实现图片的渐入。

试管中的动画

试管中的动画较为复杂，使用到的方法是首先创建若干个粒子对象，每个对象在初始化时将随机分配粒子图片和粒子大小。

对于粒子的运动动画，总共维护了6个变量：粒子当前坐标（x坐标、y坐标）；粒子当前速度（x方向，y方向）；粒子的旋转角度和旋转速度。

在每次更新，调用update方法时，首先给三个速度变量加上随机加速度值，再把速度加到对应的坐标、旋转角度变量上，即可实现较为随机的运动，为了防止随着时间推移运动速度逐渐失去控制，可以对速度变量上下限进行限制。当粒子向上运动至超出上届时可以将其坐标直接置为下届。

流动腔和激光的动画

流动腔内细胞流动就较为简单。蓝色背景和粒子图片每次更新时向下移动若干像素，在到达下届后，蓝色背景停止移动，而粒子图片重置为上届坐标即可。

当对应粒子运动到指定y范围内时，显示激光的动画即可。

文本显示

直接获取当前鼠标x，y坐标，在暴力判断范围后显示对应的文本即可。
